iPhoneBeacon
============

【ヒカ☆ラボ】あなたのMacがあっという間にBeacon端末に変身！～2015年必ずクル「iBeacon」を1から10まで～

#### 【対応端末】
動作確認を行ったものはiPhone5のみです。それ以降の端末は動作すると思います。
　
#### 注意！
アプリの実行前に、BluetoothをONにしてください。
